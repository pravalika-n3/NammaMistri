# Namma-Mistri Android App

Kotlin Android app for rural construction site management. It is built with Jetpack Compose and follows the SOP guide in this folder.

## Features

- Manage construction sites with owner, location, and phone details.
- Calculate bricks, cement bags, and sand loads for walls and rooms.
- Track workers, daily wages, days present, advances, and balance due.
- Maintain local material rates.
- Add site progress photo references from the device gallery.

## Open In Android Studio

1. Open this folder in Android Studio.
2. Let Gradle sync download the Android Gradle Plugin and Compose dependencies.
3. Run the `app` configuration on an emulator or Android device.

The package name is `com.example.nammamistri`.
